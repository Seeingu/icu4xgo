# icu4xgo

This is a Go binding for the ICU4X project, which provides internationalization capabilities for software applications.

## Installation

To install the necessary dependencies, you need to have Cargo and CMake installed on your system.

### Prerequisites

- [Cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html): The Rust package manager.
- [CMake](https://cmake.org/install/): A cross-platform build system.

### Build

To build the project, use the following command:

```sh
make build
```
